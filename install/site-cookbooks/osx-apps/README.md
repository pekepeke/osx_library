osx-apps Cookbook
=================
This cookbooks setups the machine.

Requirements
------------

#### platform
- `mac osx`

Usage
-----
#### osx-apps::default

Just include `osx-apps` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[osx-apps]"
  ]
}
```
